How To Use

Creating account

Even Though We provide Free Service, It is Required that You have an Iris Account in Order to
Get your access to your API ID. To Create an Iris Account Click on the "Free Sign Up" Link in the
Navbar Fill the Form and then after Submmiting it you Will see the confirmation Of your Registration
along with Your API ID, And if you forget to notice that don't Worry We've got You Covered !

Checking API Id

You forget to notice the API Id During Registration Process don't Worry you can follow these:
Get your access to your API ID. To Check Your Iris API Id, Click on the "Check API ID" Link in the
Navbar Fill the Form and then after Submmiting it you Will see Your API ID

Using API ID

Using the API ID is Quite Easier, In Our Case, Our Main Program is inside a Function
named 'respond'. Now the skeleton should look like "https://website.com/respond/api_id/query"
("Website.com--->Website Name"; "respond--->Name of the function"; "api_id--->Your API ID";
"query--->Your Query")
